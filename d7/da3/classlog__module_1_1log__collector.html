Detailed Description

A class for collecting and managing log messages.

This class inherits from thread_base and provides functionality for collecting, processing, and distributing log messages to console and file outputs. It manages separate queues for log collection, console output, and file output. The class supports various string types for log messages including std::string, std::wstring, std::u16string, and std::u32string.
