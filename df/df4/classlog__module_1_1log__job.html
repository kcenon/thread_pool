Detailed Description

Represents a logging job derived from the base job class.

This class encapsulates the functionality for creating and executing logging operations as jobs within the job system. It supports different log types and optional start times for more detailed logging. The class handles various string types (std::string, std::wstring, std::u16string, std::u32string) to accommodate different character encodings.
