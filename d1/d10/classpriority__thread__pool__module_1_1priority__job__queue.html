Detailed Description

template<typename priority_type>
class priority_thread_pool_module::priority_job_queue< priority_type >

A queue for managing priority jobs.

This template class implements a queue that manages jobs with different priority levels. It inherits from job_queue and std::enable_shared_from_this to allow creating shared_ptr from this.

Template Parameters
priority_typeThe type used to represent the priority levels.
