Detailed Description

A utility class for string conversions and manipulations.

This class provides static methods to convert between std::string, std::wstring, std::u16string, and std::u32string. It also includes methods for converting between strings and byte arrays, and for string splitting operations. Additionally, it provides a constant for the UTF-8 BOM.
