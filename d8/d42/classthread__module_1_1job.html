Detailed Description

Represents a job that can be executed by a job queue.

This class is a base class for all jobs that can be executed by a job queue. It provides a callback function that is executed when the job is executed. The callback function must return a tuple with a boolean value indicating whether the job was executed successfully and an optional string with an error message in case the job failed.
